# Contributing

Obrigado pelo interesse em contribuir! Este repositório é voltado para estudos, mas contribuições são bem-vindas.

## Como contribuir

1. **Fork** o repositório
2. Crie uma **branch** para sua feature: `git checkout -b feat/minha-feature`
3. Faça seus **commits** seguindo o padrão [Conventional Commits](https://www.conventionalcommits.org/)
4. Envie um **Pull Request**

## Padrão de commits

| Tipo | Descrição |
|---|---|
| `feat` | Nova funcionalidade ou módulo |
| `fix` | Correção de bug |
| `docs` | Alteração em documentação |
| `refactor` | Refatoração sem alteração de funcionalidade |
| `test` | Adição ou modificação de testes |
| `chore` | Tarefas de manutenção (CI, configs, etc.) |

## Executando testes

```bash
cargo test
```

## Pré-requisitos

- Rust 1.85+
- Cargo
- Docker (opcional)
